# CS361 HW5 - Automata Theory

## Assignment Details

**Course:** CS361  
**Term:** Fall 2025  
**Total Points:** 30 (15 points per question)

## Questions

### Question 1 (15 points)
Use the pumping lemma to prove the following language is not regular:  
L1 = {w over {a,b,c} | w has twice the number of a's as b's.}

**Grading Criteria:**
- Correct pumping lemma setup
- Valid contradiction
- Proper conclusion

### Question 2 (15 points)
Prove the language L2 defined below is either regular or not regular.  
L2 = {w over {a,b} | w has the same number of a's and b's, and |w| <= 2}

**Grading Criteria:**
- Correctly determined regularity
- Provided proper proof
- Enumerated language or provided regex/DFA

## Grading Rubric

Per question (15 points each):
- **No submission:** 0 points
- **Attempted:** 7.5 points (50% - partial understanding shown)
- **Mostly correct:** 14 points (minor errors or missing steps)
- **Fully correct:** 15 points (complete and accurate solution)
